# **四、Node.js 环境**

1. **Node.js 基础**
   - 安装与运行
   - 模块系统：核心模块、第三方模块、自定义模块

2. **文件系统**
   - 文件读写：`fs.readFile`、`fs.writeFile`
   - 同步与异步操作

3. **HTTP 服务**
   - 创建服务器：`http.createServer()`
   - 路由与请求处理

4. **NPM 包管理**
   - 安装与卸载包：`npm install`、`npm uninstall`
   - 项目初始化：`npm init`
   - 脚本运行：`npm run script`
