### 使用示例

```javascript
// 测试 myPromiseAll
myPromiseAll([Promise.resolve(1), Promise.resolve(2)])
  .then(console.log); // [1, 2]

// 测试 myPromiseAny
myPromiseAny([Promise.reject(1), Promise.resolve(2)])
  .then(console.log); // 2

// 测试 myFinally
Promise.resolve(42).myFinally(() => console.log('done'))
  .then(console.log); // logs: 'done', then 42
```
