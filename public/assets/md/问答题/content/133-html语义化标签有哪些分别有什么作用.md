## html语义化标签，有哪些，分别有什么作用
以下是 HTML 常见语义化标签及其作用的表格总结：

| 标签 | 作用说明 |
|------|----------|
| `<header>` | 表示页面或区块的头部，常包含标题、logo、导航等 |
| `<nav>` | 定义导航链接区域，如主导航菜单 |
| `<main>` | 表示页面的核心内容，每个页面应仅使用一次 |
| `<article>` | 表示独立、可复用的内容单元（如文章、评论、帖子） |
| `<section>` | 表示文档中的一个主题性区块，通常带标题 |
| `<aside>` | 表示与主内容间接相关的内容，如侧边栏、广告、引用 |
| `<footer>` | 表示页面或区块的尾部，常含版权、作者、链接等 |
| `<figure>` | 包裹图片、图表、代码等自包含内容 |
| `<figcaption>` | 为 `<figure>` 提供标题或说明文字 |
| `<time>` | 表示日期或时间，支持 `datetime` 属性提供机器可读格式 |
| `<mark>` | 高亮显示文本，用于强调或标记关键词 |
| `<details>` | 创建可折叠的内容区块 |
| `<summary>` | 作为 `<details>` 的可见标题，点击可展开/收起内容 |
| `<address>` | 提供作者或组织的联系信息，通常用于页脚 |
