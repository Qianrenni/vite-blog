### 总结

| 语言     | 主要 GC 类型        | 关键优化手段                          |
|----------|---------------------|----------------------------------------|
| Java     | 分代 + 并发/并行 GC | JVM 调优、对象池、减少装箱            |
| Go       | 并发三色标记        | sync.Pool、避免逃逸、pprof 分析       |
| C#       | 分代 + LOH          | ArrayPool、Span、服务器 GC            |
| Python   | 引用计数 + 分代 GC  | 生成器、weakref、调整 gc 阈值         |
| JS       | 标记-清除           | 避免闭包泄漏、事件清理、DevTools 分析 |
