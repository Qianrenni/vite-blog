### 一、HTTP 与 HTTPS 的核心区别

| 特性 | HTTP | HTTPS |
|------|------|--------|
| 全称 | HyperText Transfer Protocol | HyperText Transfer Protocol Secure |
| 端口 | 默认 80 | 默认 443 |
| 数据传输 | 明文传输（不加密） | 加密传输（使用 SSL/TLS） |
| 安全性 | 不安全，易被窃听、篡改、劫持 | 安全，防窃听、防篡改、防中间人攻击 |
| 证书 | 无需证书 | 需要由受信任的 CA（证书颁发机构）签发的 SSL/TLS 证书 |
| 性能 | 略快（无加密开销） | 略慢（加密/解密有开销，但现代优化已大幅降低影响） |
| SEO/浏览器支持 | 被标记为“不安全” | 被浏览器标记为“安全”，有利于 SEO |
