## import A from B 是拷贝还是引用

| 特性 | 说明 |
|------|------|
| **本质** | 只读的实时绑定（live read-only binding） |
| **是否拷贝值？** | ❌ 否（不是值的副本） |
| **是否可重新赋值？** | ❌ 否（`A = x` 会报错） |
| **是否实时更新？** | ✅ 是（若 B 模块内部修改了导出变量，A 能看到最新值） |
| **对象属性可改吗？** | ✅ 是（因对象是引用类型，修改属性不改变绑定本身） |
| **原始类型可改吗？** | ❌ 否（无法赋值，且原始值不可变） |
| **与 CommonJS 区别** | ESM 是“活绑定”；CommonJS 是“导出时的值快照” |
