### ✅ 方法一：使用一个遮罩层 + 一个镂空区域（推荐）

```html
<div class="overlay">
  <div class="hole" style="top: 100px; left: 200px; width: 200px; height: 100px;"></div>
</div>
```

```css
.overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100vw;
  height: 100vh;
  background: rgba(0, 0, 0, 0.6); /* 半透明遮罩 */
  z-index: 9999;
  pointer-events: none; /* 可选：允许点击穿透 */
}

.hole {
  position: absolute;
  background: transparent;
  border: 2px solid white; /* 可选：加个边框突出镂空区域 */
  box-shadow: 0 0 0 9999px rgba(0, 0, 0, 0.6); /* 关键：用超大阴影模拟遮罩 */
  /* 注意：box-shadow 的 spread 值要足够大，覆盖整个屏幕 */
  pointer-events: auto; /* 如果需要点击镂空区域 */
}
```

> ✅ 优点：简单、高性能、无需多个 DOM 元素叠加。
