### 总结对比表

| 特性 | SQL 数据库 | NoSQL 数据库 |
|------|-----------|-------------|
| 数据模型 | 表结构（行/列） | 灵活（文档、键值、列、图等） |
| Schema | 固定（需预定义） | 动态（可随时变更） |
| 扩展性 | 垂直扩展为主 | 水平扩展为主 |
| 事务支持 | 强 ACID | 弱 ACID / BASE（部分支持事务） |
| 查询能力 | 强大（JOIN、子查询） | 有限（通常无 JOIN） |
| 一致性 | 强一致性 | 最终一致性（可配置） |
| 典型场景 | 金融、ERP、传统业务系统 | 大数据、实时分析、高并发 Web 应用 |
