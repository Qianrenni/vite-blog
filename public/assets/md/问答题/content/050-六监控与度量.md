### 六、监控与度量

#### 1. 核心性能指标（Web Vitals）

- LCP（最大内容绘制）< 2.5s
- FID（首次输入延迟）< 100ms（已替换为 INP）
- INP（交互到下一次绘制）< 200ms
- CLS（累计布局偏移）< 0.1
- TTI（可交互时间）< 5s
- TTFB（首字节时间）< 500ms

#### 2. 监控工具

- Chrome DevTools Performance / Lighthouse
- WebPageTest.org
- Google PageSpeed Insights
- Sentry / Performance API / RUM（真实用户监控）

#### 3. 建立性能预算（Performance Budget）

- 限制 JS/CSS/图片总大小
- 限制请求数量
- 设定关键指标阈值，CI/CD 中自动化检测
