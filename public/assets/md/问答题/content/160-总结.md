### 总结
| 方面             | Vue 2                     | Vue 3                              |
|------------------|---------------------------|------------------------------------|
| 响应式系统       | `Object.defineProperty`   | `Proxy`                            |
| API 风格         | Options API               | Options API + Composition API      |
| TypeScript       | 支持有限                  | 原生良好支持                       |
| 性能             | 较好                      | 更快、更小                         |
| 新特性           | 无 Fragment/Teleport 等   | 支持 Fragment、Teleport、Suspense  |
| 全局 API         | 直接挂载在 Vue 上         | 通过应用实例调用                   |
