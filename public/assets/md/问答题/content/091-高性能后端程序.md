## 高性能后端程序

| 检查项                                      |
|------------------------------------------|
| 使用异步框架（FastAPI / Starlette）              |
| 网络请求用 `aiohttp` / `httpx`                |
| 文件 I/O 用 `aiofiles` 或线程池                 |
| 数据库用异步驱动（`asyncpg`, `aiomysql`, `motor`） |
| 超大表已分表（按 book_id 哈希）                     |
| 热点数据用 Redis 缓存                           |
| 压测过接口（wrk / locust）QPS > 2000            |
| 负载均衡 + 多节点部署（Nginx + 3+ 实例）              |
| 健康检查 + 自动剔除故障节点                          |
| 支持滚动更新 / 蓝绿部署                            |
| 自动扩缩容（K8s HPA）                           |
