### 六、常见面试题解析

#### Q1：以下代码输出什么？

```js
console.log('start');

setTimeout(() => console.log('A'), 0);

Promise.resolve()
    .then(() => console.log('B'))
    .then(() => console.log('C'));

console.log('end');
```

✅ **答案**：`start → end → B → C → A`

---

#### Q2：微任务中添加新微任务会怎样？

```js
Promise.resolve().then(() => {
    console.log('1');
    Promise.resolve().then(() => console.log('2'));
});
```

✅ **答案**：`1 → 2`  
→ 微任务队列是**动态清空**的，新加入的微任务也会在本轮执行完。
