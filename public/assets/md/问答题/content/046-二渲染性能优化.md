### 二、渲染性能优化

#### 1. 减少重排（Reflow）与重绘（Repaint）

- 避免频繁操作 DOM
- 批量修改样式（使用 class 替代逐条 style 修改）
- 使用 `transform` 和 `opacity` 实现动画（触发 GPU 加速，不触发重排）
- 离线操作 DOM（DocumentFragment / 虚拟 DOM）

#### 2. 优化 CSS 选择器

- 避免深层嵌套选择器
- 避免使用通配符 `*`、属性选择器等低效选择器
- 减少使用 CSS 表达式（expression）

#### 3. 使用高效的布局方式

- Flexbox / Grid 布局优于传统 float / table
- 避免使用 table 布局（重排代价高）
- 避免强制同步布局（Forced Synchronous Layouts）

#### 4. 控制渲染层级

- 减少层叠上下文（z-index）滥用
- 合理使用 `will-change` 提示浏览器优化
- 避免过度使用 `position: fixed`（移动端性能差）

#### 5. 使用虚拟列表（Virtual Scrolling）

- 长列表只渲染可视区域内容（React Virtualized / Vue Virtual Scroll）
