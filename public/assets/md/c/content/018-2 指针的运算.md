## **2. 指针的运算**

### **2.1 指针的加减运算**

- 指针可以进行加减运算，但结果取决于指针的类型。
- **规则**：
  - 指针加减一个整数时，指针会移动 `n * sizeof(type)` 字节。
  - 示例：

    ```c
    int arr[5] = {1, 2, 3, 4, 5};
    int *p = arr;  // 指向数组的第一个元素
    printf("%d\n", *(p + 2));  // 输出：3（移动两个整型大小的位置）
    ```

### **2.2 指针的比较运算**

- 指针可以使用关系运算符（`==`, `!=`, `<`, `>`, `<=`, `>=`）进行比较。
- 比较通常用于判断指针是否指向同一块内存区域或顺序。
- 示例：

  ```c
  int arr[5] = {1, 2, 3, 4, 5};
  int *p1 = &arr[0];
  int *p2 = &arr[2];
  if (p1 < p2) {
      printf("p1 points to an earlier element than p2.\n");
  }
  ```
