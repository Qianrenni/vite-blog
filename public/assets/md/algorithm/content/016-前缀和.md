## **前缀和**

### **和为K的子数组**

> **问题描述**:给你一个整数数组 nums 和一个整数 k ，请你统计并返回 该数组中和为 k 的子数组的个数 。子数组是数组中元素的连续非空序列。

```python
class Solution:
  def subarraySum(self, nums: List[int], k: int) -> int:
    map = defaultdict(int)
    map[0] = 1
    prefix = 0
    answer = 0
    for num in nums:
      prefix += num
      answer += map[prefix - k]
      map[prefix] += 1
    return answer
```
