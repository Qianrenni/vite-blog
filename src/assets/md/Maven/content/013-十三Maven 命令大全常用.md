# 🧰 十三、Maven 命令大全（常用）

| 命令 | 说明 |
|------|------|
| `mvn clean` | 清理输出目录 |
| `mvn compile` | 编译主代码 |
| `mvn test` | 执行测试 |
| `mvn package` | 打包项目（如 JAR、WAR） |
| `mvn install` | 安装到本地仓库 |
| `mvn deploy` | 部署到远程仓库 |
| `mvn dependency:tree` | 查看依赖树 |
| `mvn archetype:generate` | 创建新项目 |
| `mvn site` | 生成项目站点文档 |
| `mvn verify` | 执行集成测试 |
