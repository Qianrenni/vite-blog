## **2. 循环语句**

循环语句用于重复执行一段代码，直到满足特定条件为止。C 语言提供了三种主要的循环结构。

### **2.1 `for` 循环**

- **作用**：适用于已知循环次数的情况。
- **语法**：

  ```c
  for (initialization; condition; increment/decrement) {
      // 循环体
  }
  ```

- **示例**：

  ```c
  for (int i = 0; i < 5; i++) {
      printf("i = %d\n", i);
  }
  ```

### **2.2 `while` 循环**

- **作用**：先判断条件，再执行循环体。适用于不确定循环次数的情况。
- **语法**：

  ```c
  while (condition) {
      // 循环体
  }
  ```

- **示例**：

  ```c
  int i = 0;
  while (i < 5) {
      printf("i = %d\n", i);
      i++;
  }
  ```

### **2.3 `do-while` 循环**

- **作用**：先执行一次循环体，再判断条件。至少会执行一次循环体。
- **语法**：

  ```c
  do {
      // 循环体
  } while (condition);
  ```

- **示例**：

  ```c
  int i = 0;
  do {
      printf("i = %d\n", i);
      i++;
  } while (i < 5);
  ```

### **2.4 循环嵌套**

- **作用**：在一个循环体内嵌套另一个循环。
- **示例**：

  ```c
  for (int i = 0; i < 3; i++) {
      for (int j = 0; j < 3; j++) {
          printf("i = %d, j = %d\n", i, j);
      }
  }
  ```
