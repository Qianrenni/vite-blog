# 📊 总结对比表

| 类型         | 描述                                 | 原因                             | 解决方案                                               |
|--------------|--------------------------------------|----------------------------------|--------------------------------------------------------|
| 缓存穿透     | 查询不存在的数据                     | 数据不存在                       | 布隆过滤器、缓存空值、参数校验                         |
| 缓存击穿     | 热点 key 失效，大量并发访问          | 热点 key 过期                    | 互斥锁、逻辑过期、热点 key 永不过期                    |
| 缓存雪崩     | 大量 key 同时失效或 Redis 宕机        | key 设置相同过期时间 / Redis 故障 | 随机过期时间、集群部署、降级熔断、多级缓存             |
