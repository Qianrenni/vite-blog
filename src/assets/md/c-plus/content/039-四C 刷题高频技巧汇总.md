## 🎁 四、C++ 刷题高频技巧汇总

| 场景 | 写法 |
|------|------|
| 构造二维数组 | `vector<vector<int>> dp(n, vector<int>(m, 0))` |
| 构造哈希表 | `unordered_map<string, int> mp;` |
| 遍历字符串 | `for (char c : s)` |
| 遍历 vector | `for (int x : vec)` |
| 打印调试 | `cout << x << endl;` |
| 快速读入 | `cin >> s;` |
| 快速读入多行 | `while (cin >> s)` |
| 去重 | `unordered_set<int> st(vec.begin(), vec.end())` |
